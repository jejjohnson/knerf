# knerf

> Minimal neural fields with keras-core.

## Installation

```bash
pip install git+https://github.com/jejjohnson/knerf
```

Or for development with `uv`:

```bash
git clone https://github.com/jejjohnson/knerf.git
cd knerf
uv sync --all-groups
```

## Quickstart

```python
import knerf
```
